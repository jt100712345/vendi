# Resources Guide

This folder is for stable, evergreen reference materials and external knowledge.

**Guidelines:**
- **External Links:** Store links to external articles, documentation, videos, or other online resources. Consider using web capture tools to save readable versions locally.
- **Personal Summaries:** For key resources, create a summary note with your own insights and reflections.
- **Organize by Type/Topic:** Use sub-folders for different types of resources (e.g., `Books/`, `Articles/`, `Tutorials/`) or by overarching topics.
- **Metadata:** Include useful metadata such as source, author, date accessed, and a brief description.
- **Tagging:** Use tags for easy categorization (e.g., `#docs`, `#api-reference`, `#book-summary`, `#tool-guide`).
- **Avoid Duplication:** Link to existing notes rather than duplicating content.
- **Maintain Quality:** Periodically review resources to ensure they are still relevant and accessible.
- **Attachments:** For local files (e.g., PDFs, images), ensure they are placed within this folder or a relevant sub-folder and linked from notes.
