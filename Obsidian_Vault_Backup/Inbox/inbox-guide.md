# Inbox Guide

This folder is for new, raw, or unclassified information that needs to be processed.

**Guidelines:**
- **Capture:** Quickly drop any new ideas, notes, links, or fleeting thoughts here.
- **Processing:** Periodically review and process items:
    - **Classify:** Move to `Projects`, `Thinking`, or `Resources`.
    - **Refine:** Elaborate on short notes.
    - **Tag:** Use relevant tags for easy retrieval.
- **Naming Convention:** Use `YYYYMMDD-HHMM-short-description.md` for quick captures, or descriptive names for more structured inputs.
- **Linking:** Create internal links to existing notes as soon as connections are clear.
- **No Orphaned Notes:** Strive to link new notes to existing knowledge or define their place in the vault.
